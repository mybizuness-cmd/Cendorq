# Cendorq Visual Command Quality Standard

Visual quality is part of category ownership. Cendorq should feel controlled, premium, clear, useful, and serious within seconds.

## Core question

Every important surface should answer: What is the safest next command?

## Required qualities

- Premium restraint
- Strong hierarchy
- Immediate scannability
- Clear proof sequence
- One dominant next action
- Mobile clarity
- Serious business tone
- Report readability
- Dashboard command clarity
- Clean visual focus

## Surface review

Public pages should make the buyer path obvious. Report surfaces should separate score, finding, proof, limitation, next move, and repair priority. Dashboard surfaces should show state, next action, report access, support path, billing path, and current control posture.

## Mobile review

Mobile screens must preserve the main action, clear hierarchy, readable cards, and enough spacing for trust.

## Release review

Before a major customer-facing surface is ready, confirm that the main action is obvious, the hierarchy is restrained, mobile is clear, and the design makes Cendorq feel more trusted rather than louder.
